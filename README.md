#FreePC
Login restriction management system.

##What is FreePC?

##Requirements
###To use FreePC, you need to install:
  - python (2.7)
  - python-dateutil
  - django (>=1.6.5)
  - a web server (apache, lighttp, ngynx)

###Install using pip:
  - [djangorestframework][rest]
  - markdown
  - django-filter
  - PyYAML
